modal logic

Propositional logic classifies propositions into main categories: true and false. Modal logic uses a finer classification and considers further modes in which a proposition may be true and false. Not only is the proposition that there are infinitely many primes true, it is necessarily true. On the other hand, the proposition that snow is white is only possibly true. These modalities play a crucial role in argumentation in logic and philosophy.

Consider, first, an argument for the existence of numbers:

Numbers can exist necessarily if they can exist at all. Numbers can exist. Therefore, numbers can exist necessarily. It follows that numbers do, in fact, exist.

While propositionally invalid, the argument retains a measure of plausibility, which we can explain by appeal to these modalities and how they are related to the original two categories and to each other.

  • Why is the argument above propositionally invalid?

In modal propositional logic, we supplement the vocabulary of propositional logic with a sentential operator ◻​​, read: ‘it is necessary that’, and ◊​​​, read: ‘it is possible that’. So, if we let N​ stand for the proposition that numbers exist, we may formalize the argument above as follows:

  1. ◊N→◊◻N​
  2. ◊N​
  3. ◊◻N
  4. N​

Once we regiment the argument in the language of modal propositional logic, we may ask how exactly to assess its validity. Does, in other words, the conclusion follow from the premises? Notice that it will not do to treat the new sentential operators as truth-functional operators.

Aristotle’s Sea Battle Argument is another case in point:

If a thing is white now, it was true before to say that it would be white, so that of anything that has taken place, it was always true to say that ‘it is’ or ‘it will be’. But if it was always true to say that a thing is or will be, it is not. possible that that it should not be or not be about to be, and when a thing cannot not come to be, it is impossible that it should not come to be, and when it is impossible that it should not come to be, it must come to be.

  1. If there is a sea battle tomorrow, then necessarily, there will be a sea battle tomorrow
  2. If not, then, necessarily, there will not be one.
  3. Now, either there will be a sea battle tomorrow or there will not be one.
  4. Therefore, either, necessarily, there will be a sea batttle tomorrow, or, necessarily, there will not be one.

Here is a candidate formalization of the argument, where B stands for the proposition that there will be a sea battle tomorrow:

  1. B→◻B
  2. ¬B→◻¬B
  3. B∨¬B
  4. ◻B∨◻¬B

On this formalization, the argument appears to be propositionally valid, but you may wonder whether it is sound. That would in turn require us to look more closely into the status of the principle: A→◻A.

To return now to the question of validity, notice that the modal operator ◻ is not a truth-functional connective: if A is true, then whether ◻A is true will be very sensitive to the choice of A, e.g., ‘necessarily, two is a prime number’ is true, but ‘necessarily, snow is white’ is not true despite the fact that both ‘two is a prime number’ and ‘snow is white’ are both true. So, there is no hope to generalize the usual truth-functional account of propositional validity to cover the case of modal propositional logic.

possible worlds

One basic semantic framework for modal logic makes use of possible worlds models, which consist of a non-empty set of possible worlds W, an accessibility relation R on that set, and an assignment V​ of truth values to sentence letters of the language at possible worlds.

Often possible worlds models are drawn instead of formally specified. The diagram below corresponds to a possible worlds model, where W={w1,w2,w3,w4}​, R={⟨w1,w2⟩,⟨w1,w3⟩,⟨w2,w4⟩,⟨w4,w3⟩}​, and V(p,w3)=T,V(p,w4)=T,V(q,w2)=T,V(q,w4)=T​​​ with other sentence letters assigned F​ at possible worlds in the model.

 

Complex sentences are evaluated as true or false in a possible world according to standard rules of evaluation. The crucial clause concerns sentences of the form ◻φ and ◊φ, which are evaluated as follows:

  • ◻φ is true at a world w in the model if, and only if, φ is true at every world w′ accessible from w​.
  • ◊φ is true at a world w in the model if, and only if, φ is true at some world w′ accessible from w.

For example, in the model above:

  • ◻(p∨q) is true at w1 because p∨q is true at every world accessible from w1, namely, w2 and w3.
  • ◊(p∧q) is true at w2 because p∧q is true at some world accessible from w2, namely, w4​.

At each world, propositional connectives are evaluated as usual.

One crucial payoff of possible worlds model theory lies in the correspondence between the validity of certain modal principles and the fact that the accessibility relation satisfies certain structural features. To be more precise, a formula φ​ is valid in a model iff φ​​​ is true at each world in the model. In particular:

  • ◻φ→φ is valid in every possible worlds model in which the accessibility relation is reflexive, i.e., every world is accessible from itself.
  • ◻φ→◊φ is valid in every possible worlds model in which the accessibility relation is serial, i.e., every world is related to another world by the accessibility relation.
  • ◻φ→◻◻φ​​​ is valid in every possible worlds model in which the accessibility relation is transitive, i.e., if a world is accessible from another, then the former is accessible from any world from which the latter is accessible.
  • φ→◻◊φ is valid in every possible worlds model in which the accessibility relation is symmetric, i.e., if a world is accessible from another, then the latter is accessible from the former.

This means that the question of what modal principles are true is connected to the question of what are reasonable structural constraints on the relation of relative possibility understood as a binary relation on the domain of possible worlds. So, for example, the validity of the first argument we considered above is tied to the question of whether relative possibility should be conceived as a symmetric relation.

quantified modal logic

Matters are more complicated when we combine quantification and modality. One thought at this point is just to expand the logic of quantification with whatever modal principles we find reasonable. The problem is that unless we are careful, we may find ourselves in an uncomfortable position. For consider the argument:

1x=xReflexivity of Identity2∃y x=yExistential Generalization3◻∃y x=yNecessitation4∀x◻∃y x=yUniversal Generalization

You may be tempted to read the conclusion as the claim that everything exists necessarily, which seems plainly false. Very closely aligned arguments provide proofs of further modal principles:

◻∀x◻∃y x=yThe Necessary Necessity of Existence◻∀xφ→∀x◻φThe Converse Barcan Formula

On the face of it, the first principle says that necessarily, everything exists necessarily, which is apparently unpalatable. When we help ourselves to the schema φ→◻◊φ, we are able to deduce every instance of the principle:

∀x◻φ→◻∀xφThe Barcan Formula

or, equivalently:

◊∃xφ→∃x◊φ

If it is, for example, possible for there to be immaterial beings, then something is possibly an immaterial being. But this again seems a mistake. The antecedent merely states the possible existence of something with a certain feature, whereas the consequent asserts the existence of something possibly exemplifying the feature in question. The fact that these results flow from the interaction of basic quantificational axioms and elementary modal principles has suggested to many that we should rethink the foundations of quantificational logic.
